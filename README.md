# 2. Summary

**Purpose:**  
This dataset examines how digital and AI-powered tools influence cognitive load, mental fatigue, decision offloading, and the after-effects of AI interaction.

**Scope:**  
Adults aged 18+, across the UK, US, Australia, New Zealand, and Ireland.  
Data collected November 2025 via Prolific.

**Themes:**  
Cognitive load · Mental fatigue · Decision offloading · AI reliance · Agency · Digital overstimulation · Mental after-effects.

**Outputs:**  
Numeric Likert-style scores (1–7) plus open-text reflections.

**Data Integrity Note:**  
This dataset has been fully cleaned, anonymised, and verified according to HCI’s open-data protocol.  
A SHA-256 checksum file is included to ensure long-term data integrity.  
Any future updates will be released as new versions.

# 7. Method

**Design:**  
Cross-sectional online survey.

**Instrument:**  
19 × Likert items (1–7)  
1 × open-text item  
Demographic questions (age, gender, work status, country, AI-tool use)

**Sampling:**  
Prolific.  
Quotas: UK, US, AU, NZ, IE.  
Collection period: November 2025.

**Ethics:**  
Informed consent obtained.  
Data anonymised.  
Participants could withdraw prior to submission.

# 9. Processing & Cleaning

**Pipeline:**  
1. Exported raw responses from Google Forms  
2. Verified no missing mandatory responses  
3. Checked for duplicates (none found)  
4. Checked for straight-lining (none detected)  
5. No encoding issues present  
6. Verified clean numeric ranges (1–7)  
7. Renamed headers to snake_case  
8. Saved final cleaned dataset  
9. Prepared README + data dictionary  
10. SHA-256 checksums generated after publication
